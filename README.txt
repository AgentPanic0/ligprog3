Compilacao:
<g++ -Wall -std=c++11 main.cpp catalogo.cpp -o execTrab2> ou <make all> ou <make execTrab3>

Execucao:
<.\execTrab3 <nomeArquivo.txt> >
nomeArquivo.txt = bancoDados.txt

Utilizacao:
Programa intuitivo basta ler e interagir, digitando no teclado e pressionado enter.
os nomes dos campos sao: nome,produtora,nota
o arquivo a ser passado como argumento deve esta no formato descrito acima sem linhas vazias.
